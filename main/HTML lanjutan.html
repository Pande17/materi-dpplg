APA ITU HTML?

HTML

HTML (Hypertext Markup Language) adalah bahasa pemrograman dasar yang digunakan untuk membangun struktur dasar halaman web. Bahasa HTML adalah bahasa yang sangat diminati banyak orang dikarenakan sangat mudah untuk dipelajari oleh orang-orang yang baru masuk ke dunia pemrograman. Banyak juga website yang menggunakan bahasa pemrograman HTML ini. Dalam HTML, informasi diorganisir dalam bentuk elemen-elemen yang dikenal sebagai "tag," yang mengelompokkan dan mendefinisikan konten seperti teks, gambar, tautan, dan banyak lagi. Setiap tag memiliki peran khusus dalam menentukan tampilan dan struktur halaman web. Misalnya, tag <.p> digunakan untuk paragraf teks, sementara tag <.img> digunakan untuk menampilkan gambar. HTML juga memungkinkan Anda untuk membuat tautan antara halaman web dan menyusun hierarki dengan menggunakan tag seperti <.head> dan <.body>. Ini adalah dasar yang penting untuk memahami cara membangun halaman web dan menjadi fondasi untuk pemahaman lebih lanjut tentang pengembangan web.

APA ITU XAMPP?

Software web server apache yang di dalamnya sudah tersedia database server mySQL dan support PHP Programming

Kenapa Namanya XAMPP?

(Exe) = Cross Platform, adalah kode penanda program (software) yang dapat berjalan di banyak system operasi, XAMPP itu bisa di jalankan di banyak OS (operating system) seperti windows, mac OS, dan Linux.
A = Apache, adalah aplikasi web server yang bersifat gratis dan bisa dikembangkan oleh banyak orang (open source). Fungsi Apache pada XAMPP yaitu untuk menghasilkan halaman web yang benar bagi user berdasarkan kode PHP yang ditulis oleh pembuat web.
M = MySQL / MariaDB, merupakan aplikasi database server yang dikembangkan oleh orang yang sama. MySQL berperan dalam mengolah, mengedit, dan menghapus daftar melalui database.
P = PHP, huruf inisial untuk menunjukkan eksistensi bahasa pemrograman PHP.
P = Perl (People of relational labor), bahasa pemrograman yang di gunakan untuk segala keperluan. Bahasa pemrograman ini sudah berkurang karena munculnya Bahasa-Bahasa pemrograman baru dan tren teknologi yang berubah


Apache : Di dalam XAMPP, Apache digunakan untuk menjalankan dan mengelola situs web atau aplikasi web lokal Anda.
MySQL : MySQL adalah sistem manajemen basis data relasional (RDBMS) yang digunakan untuk menyimpan dan mengelola data dalam bentuk tabel yang terstruktur. Dalam XAMPP, MySQL digunakan sebagai basis data untuk aplikasi web. Kalian dapat membuat, mengelola, dan mengakses basis data melalui antarmuka komando atau alat manajemen basis data visual.
FileZilla: FileZilla adalah perangkat lunak FTP (File Transfer Protocol) yang memungkinkan Anda untuk mentransfer file antara komputer Anda dan server melalui jaringan. Dalam XAMPP, FileZilla digunakan sebagai alat untuk mengelola file yang terkait dengan situs web atau aplikasi web. Kalian dapat mengunggah, mengunduh, dan mengelola file di server lokal Anda.
Mercury: Mercury adalah server mail yang dapat digunakan untuk mengirim dan menerima email dalam pengaturan pengembangan lokal. Ini mencakup SMTP (Simple Mail Transfer Protocol) dan POP3 (Post Office Protocol 3).
Tomcat: Server aplikasi web yang mengimplementasikan spesifikasi Java Servlet dan JavaServer Pages (JSP). Dalam XAMPP, Tomcat dapat digunakan untuk mengembangkan dan menguji aplikasi web berbasis Java EE (Enterprise Edition).
